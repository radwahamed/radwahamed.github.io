Project Idea:

Midas is Human Computer Interaction (HCI) is an evolving area of research. It develops new ways through which people interact with computers in an easy and more natural way. Throughout the history of computing, people have been dependent on devices through which they could interact with computers. A new trend is the vision-based interaction methods, like hand gestures. Midas uses a depth camera to detect human gestures, recognize them, and trigger actions based on the type of gesture. Midas will enable developers, with very minimal experience in the Computer Vision and Artificial Intelligence fields, to develop gesture-smart systems.

Midas comes in the form of an Application Program Interface (API). It’s a library that developers would use in their own applications, and a web application to provide a sample of how our API works. In Midas, we implement three main modules: Detection, Tracking, and Recognition. These three modules will implement Computer Vision and pattern analysis technologies to help achieve the functionality in the three modules.

This system is made up of:

  • Data collection tool, which will facilitate gathering the dataset for training and testing.
  • The detection and recognition engine, which contains the software implementations to perform hand detection, hand tracking, and gesture recognition.
  • An application that calibrates user’s hand position, then uses the trained classifiers to detect and respond to hand gestures by triggering the corresponding actuator.

I was responsible for the Gesture Recognition Module in this project
